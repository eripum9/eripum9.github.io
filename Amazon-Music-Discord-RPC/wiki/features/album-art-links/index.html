Album Art & Listen Links

Artwork and listen buttons make the Discord presence feel complete instead of text-only.

What it does

Uses Amazon Music artwork first when enhanced metadata provides it. Fallback lookup can search external sources when artwork is missing.

Listen buttons

The listen button points to Amazon Music by default. Deezer remains available as an optional source if you prefer it.

Bring your own image

Custom album artwork can be configured for difficult albums, alternate album names, or local corrections when public metadata sources are wrong.

What to check

  • If artwork is wrong, check whether the track matched the wrong album.
  • If a listen button is missing, verify the selected button source and Discord visibility from another account.
  • If fallback lookup is used, exact title and artist naming matters.

Privacy notes

Fallback artwork or track matching can send track and artist search terms to lookup providers.
