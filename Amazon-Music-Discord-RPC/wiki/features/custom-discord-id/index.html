Custom Discord Application ID

Most users should keep the default Discord application ID. Custom mode is for users who want their own Discord app assets.

What it does

Lets you replace the default Discord application ID with one from your own Discord developer application.

When to use it

Use it only if you understand Discord developer applications and want custom branding or custom image assets.

How to configure it

Open Settings, switch Discord Client ID mode to Custom, and paste a valid application ID. Switch back to Default if anything breaks.

What to check

  • The ID must be numeric and valid.
  • Your Discord application assets must be configured correctly.
  • Buttons and images can behave differently with custom apps.

Privacy notes

The custom ID changes the Discord Rich Presence application used for display. It does not change what track metadata the app reads.
