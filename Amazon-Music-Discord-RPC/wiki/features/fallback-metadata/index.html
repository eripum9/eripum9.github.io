Fallback Metadata

Fallback metadata keeps basic playback working when enhanced metadata is disabled or unavailable.

What it does

Uses Windows System Media Transport Controls as the baseline source. Optional notification fallback can add better artist or album details from Amazon Music notifications.

When to use it

Use fallback-only mode if you do not want enhanced metadata, or as a backup when Amazon Music cannot be attached.

How to configure it

Turn off Enhanced Amazon metadata in Settings. Enable notification fallback only if you want Amazon Music notifications used locally for extra metadata.

What to check

  • Amazon Music must expose media metadata to Windows.
  • Notification fallback needs Amazon Music notifications enabled.
  • Amazon Music usually needs to be minimized for notifications to appear.

Privacy notes

Notification fallback reads Windows notifications locally and only uses Amazon Music notification text.
