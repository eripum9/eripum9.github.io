Last.fm and ListenBrainz Scrobbling

Scrobbling is optional. It sends listening history to Last.fm or ListenBrainz only when you enable it.

What it does

Sends now playing updates and eligible scrobbles to the selected service after enough of a track has played.

How to configure it

Open Settings. Authenticate Last.fm with the provided button, or paste and validate a ListenBrainz user token.

Clear tokens

Use Clear Scrobbling Tokens in Settings when you want to disconnect both services and start over.

What to check

  • Reconnect Last.fm if it shows as not authenticated.
  • Paste a fresh ListenBrainz token if validation fails.
  • Private session can pause scrobbling while it is enabled.

Privacy notes

Tokens are stored locally in app config. Diagnostics and log views redact known token values, but treat config files as private.
