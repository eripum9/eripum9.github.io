Startup, Tray, and Settings

Amazon Music RPC is designed to stay out of the way while giving you a proper settings surface when you need it.

Tray behavior

The app runs in the system tray. Use the tray menu to open Settings, Diagnostics, repair metadata, report issues, or quit.

Startup option

Enable start on Windows startup if you want Discord presence ready whenever you log in.

Settings UI

Settings use a WebView2-style window with dark styling, grouped controls, and automatic saving for supported controls.

Saved window sizes

Settings and diagnostics windows remember resized dimensions so they reopen at the size you chose.

What to check

If Settings does not open, make sure Microsoft Edge WebView2 runtime is installed and check Diagnostics logs if available.
