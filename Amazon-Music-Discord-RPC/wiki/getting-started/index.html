Getting Started

Amazon Music RPC gives Amazon Music on Windows a Spotify-style Discord Rich Presence. The normal installer is the recommended setup path.

1. Install the app

Download AmazonMusicRPC_Setup.exe from the latest GitHub release, run it, and choose whether you want shortcuts or startup behavior.

Download the latest release

2. Open the required apps

  • Start the Discord desktop app.
  • Start Amazon Music for Windows.
  • Start Amazon Music RPC from the desktop shortcut, Start menu, or tray.

3. Choose enhanced metadata

New installs ask whether to enable enhanced Amazon Music metadata. It is recommended because it gives the best track names, album art, pause state, and timer accuracy.

4. Play a song

Start playback in Amazon Music and check your Discord profile. You should see the song title, artist, album artwork, and live timer.

5. Tune optional features

Open Settings from the tray to enable Last.fm, ListenBrainz, private session, keyword filters, startup behavior, notification fallback, or a custom Discord application ID.
