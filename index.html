About this project

This repo contains a Pygame-based launcher, small games, and tools for configuring resolutions and Minecraft integration. The website is a lightweight static front page intended for GitHub Pages.

  • Scripts: assets/scripts/
  • Media: assets/images, assets/audio, video/
  • Minecraft integration: minecraft/

How to run

  1. Clone or download the repository to your Windows machine.
  2. Open a PowerShell terminal and activate the project's virtualenv if present: .\.venv\Scripts\Activate.ps1
  3. Run StartGame.bat from the repo root to launch the launcher.
